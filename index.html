Leadership Brief

项目背景与赛事愿景

海峡两岸高校黑客松联赛定位为连接大陆与台湾青年技术人才的年度创新竞技平台。赛事响应两岸科技融合发展的长期需求，致力于搭建一个公平、开放、专业的交流环境，让高校学子在实战中锤炼技术能力，在协作中增进相互了解。

赛事的核心目标是构建一个可持续的赛事生态。这不仅是一场比赛，更是一个连接高校、企业、资本与创新项目的生态系统。通过赛事，我们期望实现产学研的深度结合，为两岸科技产业持续输送具备实战能力的优质人才与具有市场潜力的早期项目。

从战略层面看，成功举办联赛将显著提升主办方在两岸青年科技社群中的品牌影响力，建立起覆盖重点高校的合作网络，并为赛事的长期商业化运营探索可行路径。

定位

以高校黑客松为抓手，建立面向大陆与台湾高校学生的长期品牌赛事。

Competition Model

三阶段赛制设计与整体流程

联赛采用线上初赛、线下复赛、线下决赛的三阶段递进赛制，兼顾赛事覆盖面、参与深度与巅峰对决的专业性。各阶段通过明确的晋级规则与时间安排紧密衔接，并为参赛选手提供信息通知、交通指引、技术答疑在内的全程支持服务。

Stage 1

线上初赛阶段

面向两岸所有高校在校生开放报名。参赛队伍在指定时间内于线上竞赛平台提交项目作品，作品需包含可运行的原型、技术文档与演示视频。

评审委员会将依据创新性、技术实现、商业潜力、团队协作四个维度进行盲审打分，筛选出晋级复赛的队伍。该阶段核心在于降低参与门槛，扩大赛事影响力。

Stage 2

线下复赛阶段

晋级队伍前往由合作高校承办的复赛站点进行线下路演与答辩。承办高校需提供满足路演、开发及评审需求的场地与基础网络支持。

各复赛站点将产生区域优胜队伍，获得晋级全国总决赛的资格。此阶段强调在地化连接与深度交流，提升赛事体验与高校参与度。

Stage 3

线下决赛阶段

所有晋级队伍汇聚厦门，进行为期2-3天的巅峰对决。除最终项目演示与答辩外，增设技术 workshop、产业交流会、投资对接等环节。

评审团由顶尖技术专家、知名投资人与产业领袖共同组成，确保评审的专业性与权威性，最终产生冠、亚、季军及多项单项奖。

Execution

执行团队与下一步计划

核心团队分工

  • 赛事总监：负责整体战略、资源统筹与最终决策。
  • 运营团队：负责赛程执行、选手服务、现场管理与后勤保障。
  • 技术团队：负责竞赛平台开发维护、线上评审系统及现场技术支持。
  • 高校关系组：负责两岸合作高校的对接、维护及学术资源整合。
  • 商务拓展组：负责赞助商、投资机构、媒体等外部合作伙伴的开拓与维护。
  • 品牌宣传组：负责赛事全渠道品牌传播、内容制作与媒体关系。

近期重点工作（未来1-2个月）

  • 方案细化与预算审批：完成内部评审，根据反馈调整细节，推动预算正式批准。
  • 首批资源对接启动：启动与3-5所重点目标高校及2-3家潜在冠名赞助商的初步接触。
  • 赛事平台原型开发：启动线上竞赛平台需求确认与原型设计。
  • 宣传物料基础准备：完成赛事LOGO、主视觉、宣传片脚本等核心品牌资产设计。
  • 法律与风险预案：完成赛事合作协议模板、免责条款及各类风险应对预案起草。

风险控制要点

已初步识别政策风险、资金风险、执行风险三大类。应对策略包括：密切跟进两岸交流政策导向，确保赛事合规性；采取分阶段、多元化的合作商筹措方案，避免依赖单一来源；制定详细的应急预案，覆盖天气、技术故障、公共卫生等突发情况。

政策背景与资源对接

政治契机

国民党主席郑丽文2026年4月率团访问大陆，国共两党智库论坛达成15条共同意见，为两岸机制化交流注入新动能

机制化青年交流平台

建立国共两党青年双向交流机制化平台。全国青联与国民党青年事务发展委员会定期举办两岸青年交流活动，全国青联等大陆有关单位每年邀请岛内20个青年团组来大陆交流参访，为赛事形成稳定的青年动员与后续转化通道。

两岸青年创业基金

结合两岸青年创业基金，利用现有的两岸青年创业政策补贴，方便对岸学生来大陆创业，增进彼此了解

厦门"AI产业+"政策与高层参与

厦门预计在2026年6月正式落实"AI产业+"政策，结合市政府资源可提供有力协助；同时邀请国民党青年团代表、大陆台办领导出席开幕式，提升赛事权威性和媒体关注度。

采用"三阶段配对"模式，在随机性与匹配度之间寻求平衡：

  • • 第一阶段：线上初赛期间，基于技能标签智能推荐
  • • 第二阶段：入围公布后，组织线上"速配会"双向选择
  • • 第三阶段：现场报到后，设置"24小时试配期"

具体风险场景与应对措施

技术故障风险

风险场景
  • • 网络中断
  • • 云服务故障
  • • 设备损坏
  • • 系统崩溃
应对措施
  • • 备用5G CPE自动切换，本地离线开发环境
  • • 多云备份（阿里云双活），关键数据实时同步
  • • 现场备用设备库（关键设备2倍冗余），2小时快速更换
  • • 降级方案：纸质评分+事后补录，确保评审继续

责任人：技术组 | 响应时间：5分钟-2小时

天气变化风险

风险场景
  • • 台风/暴雨
  • • 极端高温（>35℃）
  • • 大雾（影响航班）
应对措施
  • • 室内场馆备选方案（厦门佰翔会展中心），航班改签协调
  • • 加强空调保障（备用移动空调），设置清凉休息区
  • • 启动高铁备选方案（深圳/福州中转），调整开幕式时间

预警机制：提前72小时-12小时气象预警

参赛人数不足风险

风险场景
  • • 台湾团队招募不足（<30支）
  • • 整体规模不足（<60支）
  • • 现场缺席率高（>20%）
应对措施
  • • 启动备用邀请名单，放宽企业团队限制（毕业年限延长至5年）
  • • 调整赛制为"精英邀请赛"，提升单队资源投入
  • • 启动候补名单，现场开放观摩名额填充气氛

预警指标：5月15日报名数据<20支，6月1日入围确认<50支

两岸政治敏感风险

预防层
  • • 所有材料经两岸双方审核
  • • 敏感表述统一口径
  • • "九二共识""两岸同属一中"等表述规范
响应层
  • • 设立新闻发言人，统一对外口径
  • • 暂停直播，封闭现场，等待指示
替代层
  • • 预留线上完全替代方案
  • • 48小时线上黑客松+云端评审+视频颁奖

公共卫生风险

风险场景
  • • 传染病疫情
  • • 食品安全事件
  • • 突发疾病
应对措施
  • • 备用口罩、消毒用品，临时隔离区，与疾控中心建立联动
  • • 供应商资质审核，留样检测，应急预案（定点医院绿色通道）
